
## About Pizza Order APP

The Pizza order App lets you order pizzas from top curated menu . The app has payment processor(stripe). The app is robust , designed to be scalable and is behind an Apache 2 server. 

## Tech Stack

PHP
REACT
Mysql
Apache 2

## For First Timer

cd project/
composer install
npm install
php artisan serve
